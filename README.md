## Dependencias

El proyecto cuenta con las siguientes dependencias:

- React
- React-router
- React-icons
- Firebase
- Sass

## Carpetas y Archivos

El proyecto esta separado en 6 carpetas:

- scss
- components
- firebase
- pages
- assets
-context

## Estilo de escritura

- PascalCase : para los componentes y estilos.
- camelCase: para funciones.
